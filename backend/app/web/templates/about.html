About This Project

Project Overview

Flu Forecast Hub is a comprehensive data analytics project that demonstrates end-to-end data science skills including data collection, analysis, machine learning, visualization, and web development.

Technologies Used

Backend
  • Python 3.11
  • Flask (Web Framework)
  • Pandas & NumPy
  • Statsmodels (ARIMA)
  • Scikit-learn
Frontend & Tools
  • HTML5 & CSS3
  • Bootstrap 5
  • Tableau Public
  • GitHub Actions (CI/CD)
  • Pytest (32 tests)

Key Features

  • Data Pipeline: Automated data collection from CDC FluView API
  • Data Validation: Comprehensive data cleaning and validation
  • Forecasting: ARIMA model for time-series predictions
  • Visualization: Interactive Tableau dashboard with 5 charts
  • Testing: 32 unit tests with CI/CD pipeline
  • API: RESTful endpoints for data access

API Endpoints

Endpoint Description
/api/data Get recent flu data records
/api/forecast Get forecast predictions
/api/stats Get data statistics
